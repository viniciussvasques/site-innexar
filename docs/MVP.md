# MVP - StructurOne

## Escopo do MVP

### Funcionalidades Essenciais

#### 1. Autenticação Básica
- [ ] Login para empresa/construtoras
- [ ] Login para investidores
- [ ] Registro de novos usuários
- [ ] Recuperação de senha (básico)

#### 2. Cadastro de Projetos
- [ ] Criar novo projeto/empreendimento
- [ ] Editar informações do projeto
- [ ] Listar projetos da empresa
- [ ] Visualizar detalhes do projeto

#### 3. Dashboard da Empresa
- [ ] Visão geral de projetos
- [ ] Resumo financeiro básico
- [ ] Lista de investidores
- [ ] Estatísticas simples

#### 4. Portal do Investidor
- [ ] Visualizar projetos investidos
- [ ] Acompanhar progresso do projeto
- [ ] Ver relatórios básicos
- [ ] Histórico de investimentos

#### 5. Atualizações de Obra
- [ ] Criar atualização (texto)
- [ ] Upload de fotos
- [ ] Atualizar percentual de conclusão
- [ ] Listar atualizações do projeto

#### 6. Gestão Financeira Simplificada
- [ ] Registrar entrada (receita)
- [ ] Registrar saída (despesa)
- [ ] Visualizar fluxo de caixa básico
- [ ] Relacionar transações com projetos

#### 7. Estrutura Multi-tenant Mínima
- [ ] Isolamento de dados por tenant
- [ ] Middleware de tenant
- [ ] Configuração básica de tenant

#### 8. Internacionalização Básica
- [ ] Suporte PT-BR
- [ ] Suporte EN-US
- [ ] Suporte ES-ES
- [ ] Seletor de idioma

## Priorização

### Fase 1 - Fundação (Semanas 1-2)
1. Setup do projeto e estrutura
2. Configuração do banco de dados
3. Autenticação básica
4. Estrutura multi-tenant

### Fase 2 - Core Features (Semanas 3-4)
1. Cadastro de projetos
2. Dashboard da empresa
3. Portal do investidor básico

### Fase 3 - Funcionalidades (Semanas 5-6)
1. Atualizações de obra
2. Gestão financeira simplificada
3. Upload de documentos básico

### Fase 4 - Polimento (Semanas 7-8)
1. Internacionalização
2. Testes básicos
3. Ajustes de UI/UX
4. Documentação

## Definição de Pronto

Uma funcionalidade está pronta quando:
- ✅ Código implementado e revisado
- ✅ Testes básicos passando
- ✅ Documentação atualizada
- ✅ Sem erros críticos
- ✅ UI funcional e responsiva

## Limitações do MVP

- Relatórios automáticos: versão simplificada
- Upload de documentos: apenas imagens e PDFs
- Notificações: apenas em-app (sem email)
- Analytics: básico
- Integrações: nenhuma no MVP

## Próximas Versões

Após o MVP, considerar:
- Relatórios avançados e exportação
- Notificações por email
- Dashboard de analytics avançado
- Integrações com sistemas externos
- App mobile
- API pública para integrações
